# Data-Professional-Survey-Power-BI-Project

**🎯 Project Objective**

The objective of this project was to analyze the career landscape for data professionals globally using real-life survey data. The dashboard provides an interactive view of how job titles, programming languages, and geographic locations impact salary and job satisfaction.

By visualizing these metrics, this project helps aspiring data professionals understand the "barrier to entry" and current industry standards regarding compensation and work-life balance.

**❓ Key Questions Answered (KPIs)**
What is the average salary across different data-related job titles? 

Which programming language is most favored by professionals in the field? 

How difficult do professionals find it to break into the data industry? 

Does gender significantly impact earning potential in the current market?

How do salary levels in the United States compare to other countries like India and Canada? 

What is the average level of happiness regarding salary and work-life balance? 

**🛠 My Approach**
1. Data Loading & Validation
Imported raw survey data into Power BI.

Validated the dataset size, confirming 630 unique responses. 

2. Data Cleaning and Transformation
Handling Nulls: Managed missing values in "Job Title" and "Programming Language" columns.

Standardization: Normalized salary figures into USD for global comparison.


Grouping: Aggregated various programming languages (Python, R, SQL) to identify core trends. 


Categorization: Defined five distinct levels for "Difficulty to Break into Data." 

3. Data Modeling
Organized data structures to allow seamless filtering by Country and Job Title.

Created calculated measures for Average Age (29.87) and Average Salary. 

4. Visualization Techniques

Tree Map: Distribution of survey takers by country. 


Bar Charts: Salary comparisons across professional roles. 


Donut Chart: Visual representation of industry entry difficulty. 


Gauge Charts: Sentiment scores for Work-Life Balance (5.74) and Salary Happiness (4.27). 

**💡 Project Insights**
1. Salary and Role Overview

Top Earners: Data Scientists lead in average salary, followed by Data Architects and Data Engineers. 

Gender Nuance: In this specific dataset, females earned slightly more than men (55% vs 53%).


Geographic Trends: The USA shows the highest raw salaries; however, this is closely tied to the cost of living compared to India or Canada. 

2. Technical Preferences

Python is King: Python remains the most utilized and favorite language in the data community. 

The SQL Perception: A notable portion of respondents do not categorize SQL as a primary "programming language" despite its heavy use.

3. Sentiment & Barriers

Work-Life Balance: Averaged 5.74/10, showing moderate satisfaction. 


Salary Happiness: Reported lower at 4.27/10, suggesting a desire for higher compensation. 


Entry Difficulty: 42.68% find it "Neither easy nor difficult," yet a significant portion still views it as a "Difficult" transition. 

**🏁 Final Conclusion**
This dashboard offers a comprehensive look at the realities of the data profession. While Python and Data Science roles offer the highest perceived value, the industry faces challenges with salary satisfaction. For those looking to enter the field, technical proficiency in Python and a strategic focus on high-paying roles like Data Science or Architecture remain the most effective paths.
